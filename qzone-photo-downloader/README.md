# Qzone Photo Downloader
## Requirements
- python
- selenium
    ```
    pip install selenium
    ```
- [chromedriver](https://sites.google.com/a/chromium.org/chromedriver/downloads)

## Usage
Navigate to `downloader.py`, modify the following
```python
main_user = 123456
main_pass = ''
dest_user = [123456, ]
```
to yours.
